Política de Privacidad

Última actualización: Marzo 2026

1. Información general

YT Remote ("la aplicación") es un control remoto para YouTube que permite sincronizar un dispositivo como pantalla y otro como control. Esta política explica cómo tratamos la información de los usuarios.

2. Datos que NO recopilamos

YT Remote no recopila, almacena ni transmite ningún dato personal a servidores propios. Específicamente:

  • No creamos cuentas de usuario.
  • No guardamos historial de reproducción.
  • No rastreamos la ubicación del usuario.
  • No usamos cookies de seguimiento.
  • No vendemos datos a terceros.

3. Datos almacenados localmente

La aplicación guarda en el almacenamiento local del dispositivo (localStorage) únicamente:

  • El código de sala usado más recientemente.
  • La lista de videos en la cola de reproducción.
  • Preferencias de volumen.
  • El modo seleccionado (Pantalla / Control).

Estos datos nunca salen del dispositivo y el usuario puede borrarlos en cualquier momento limpiando los datos de la app o del navegador.

4. Servicios de terceros

La aplicación utiliza los siguientes servicios externos con sus propias políticas de privacidad:

  • YouTube IFrame API (Google LLC) — para reproducir videos. Política de privacidad de Google
  • PeerJS / WebRTC — para la comunicación P2P entre dispositivos. La señalización usa servidores públicos de PeerJS. No se transmite contenido multimedia a través de estos servidores.
  • noembed.com — para obtener metadatos (título, canal) de videos por URL. Solo se envía el ID del video.

5. Comunicación entre dispositivos

YT Remote utiliza WebRTC (tecnología P2P) para la comunicación entre el dispositivo pantalla y el control remoto. Los comandos de control (play, pause, volumen, etc.) se transmiten directamente entre dispositivos sin pasar por servidores propios. Los servidores de señalización de PeerJS se usan únicamente para establecer la conexión inicial.

6. Menores de edad

Esta aplicación no está dirigida a menores de 13 años. No recopilamos conscientemente información de menores.

7. Cambios en esta política

Podemos actualizar esta política ocasionalmente. Te notificaremos mediante un aviso en la aplicación cuando haya cambios relevantes.
